# 🩺 MedGest Frontend

**MedGest** (anteriormente *Clinika*) es una aplicación web para la gestión médica de clínicas y consultorios, diseñada para facilitar el manejo de pacientes, citas, historial médico y facturación.

## 🚀 Tecnologías

- ⚛️ **React + Vite** — Interfaz moderna y rápida
- 💨 **Tailwind CSS** — Estilos con utilidad
- 🔐 **Rutas protegidas** — Simulación de login y roles
- 🧭 **React Router** — Navegación SPA

## 📁 Estructura

src/
├── pages/ # Vistas principales (Dashboard, Login, Agenda, etc.)
├── components/ # Componentes reutilizables (Header, Sidebar, etc.)
├── layout/ # Layout con Header y Sidebar centralizados
├── routes.jsx # Rutas públicas y privadas
└── App.jsx # Entrada principal



## 🧪 Funcionalidades

- 🔓 Registro e inicio de sesión (simulados)
- 📅 Agenda médica (crear, ver, eliminar citas)
- 👥 Gestión de pacientes (CRUD)
- 📋 Historial médico (estructura preparada)
- 💰 Facturación (tabla simulada, lista para PDF)
